# **App Name**: Financial Compass

## Core Features:

- Financial Overview: Dashboard overview displaying total income, expenses, and net balance.
- Transaction Categorization: Categorize transactions into predefined categories (e.g., Food, Transportation, Entertainment).
- Visual Analytics: Interactive charts and graphs illustrating income and expense distribution across categories.
- Intelligent Categorization: AI tool to categorize new transactions using learned user preferences.
- Custom Reporting Period: Allow entry for custom periods of reporting. Weekly, Monthly, Quarterly and Annualy are built in.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to inspire trust and stability.
- Background color: Light blue (#E8EAF6) for a calm, uncluttered backdrop.
- Accent color: Soft violet (#9575CD) for highlighting key data points and interactive elements.
- Font: 'Inter', a sans-serif font for clean and modern readability in both headers and body text.
- Use simple, consistent icons to represent each transaction category.
- Dashboard with clear sections for income, expenses, and net balance, using a grid-based system for balanced spacing and alignment.
- Smooth transitions when toggling between data views, such as switching from monthly to annual summaries.
